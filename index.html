欢迎访问安徽牛锕新材料科技有限公司

安徽牛锕新材料科技有限公司是一家通过ISO质量环境健康三体系认证的高新技术与“专精特新”企业。我们专注于为半导体、光电及新材料行业提供高纯电子材料核心装备与厂务供应系统的研发、设计、生产、测试及安装一体化解决方案。

我们的营业项目涵盖半导体/光电厂/材料厂的工艺机台、高纯电子材料核心装备、厂务供应系统，以及高科技厂房无尘室系统、机电系统及机台二次配工程。我们致力于成为泛半导体湿电子化学品高端智能装备系统解决方案的行业领导者。

我们的优势

化学品自动供液系统 (CDS)

应用于半导体、太阳能、液晶面板等生产领域，实现化学品的自动、安全、稳定供应。支持HF, HNO3, H2SO4, KOH, H2O2, NH4OH等多种化学品。

200L桶装CDS

双桶设计，一用一备，带出口取样功能，PLC控制，触摸屏操作。

1000L IBC-CDS

大容量供应，落地式设计，双气动泵，满足大规模生产需求。

有机溶剂CDS

针对IPA, EBR等溶剂设计，带双过滤器和前后取样功能，不锈钢滚轮。

槽车充填柜 (CCB)

实现槽罐车化学品自动卸货，带氮气测试与充填功能。

研磨液供应系统 (SDS)

专为半导体CMP工艺设计，提供高稳定性的研磨液（Slurry）原液输送、在线混配和循环供应解决方案。

Slurry原液输送系统

采用FM4910认证PVC板材，PTFE气动隔膜泵，在线WetN2保湿。

Slurry混配系统

配备Levitronix磁悬浮泵、Mettler电子秤、在线Chiller控温和H2O2滴定仪。

Slurry输送系统

集成混配、搅拌、循环、过滤、输送等功能，确保供应点品质稳定。

湿法清洗设备 (Wet Bench)

应用于集成电路制造、LED、第三代化合物半导体、先进封装等领域，提供手动、半自动及全自动的湿法清洗解决方案。

  • 典型工艺：RCA / BOE / DHF / H3PO4 / 显影 / 去胶 / KOH腐蚀等。
  • 产品特点：适用4/6/8/12寸晶圆，模块化设计，自动配液，支持IPA干燥，多Robot多任务执行。

全自动清洗机

手动清洗设备

化学品处理系统

为电子级化学品生产领域提供高纯度、高洁净度的分装、灌装、纯化等系统解决方案。

  • 化学品分装输送系统：集成酸桶传送、清洗、过滤、灌装、计量、取样等功能。
  • 全自动灌装系统：国内领先技术，实现自动开盖、灌装、拧盖、贴标、扫码，适用于光刻胶、有机溶剂等，可在10级超洁净环境下运行。
  • 电子级化学品纯化系统：国内首创G5等级纯化技术，用于生产超高纯度化学品。
